# coopers-test-analysis

Lap time analysis for the Cooper 12-minute run test — lap splits and pace deviation. Configurable target pace lines for distance goals. Bring your own data and swap in the source file.

## Setup

```bash
pip install -r requirements.txt
```

Open `main_analysis.ipynb` in VS Code or any notebook environment.

## Source file

Place your data file in the `data/` folder, named `coopers_test_YYYYMMDD.ods` (or `.xlsx`), and update `EXCEL_FILE` in the notebook to match.
